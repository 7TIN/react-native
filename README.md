# Book Reading App

A simple **Book Reading App** built with **React Native** and **Expo**.  
This app allows users to read book files seamlessly with a clean, distraction free interface.  

## Features
- Import and read book files (e.g., PDF, EPUB, or text-based formats)
- Smooth navigation with React Native + Expo
- Minimal and user friendly UI for comfortable reading
- Cross platform support (Android & iOS)

## Tech Stack
- **React Native**
- **Expo**
- **TypeScript**
